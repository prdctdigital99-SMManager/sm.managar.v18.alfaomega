⚠️ Info: Data disinkronkan ke Firebase Cloud secara real-time dan juga disimpan lokal (localStorage). Backup tetap disarankan sebagai arsip.
💾 Backup Data

Ekspor seluruh data aplikasi (anak, pelayan, absensi, pengaturan) ke dalam satu file .json. Simpan file ini di tempat aman seperti Google Drive, WhatsApp, atau email.

File akan otomatis terunduh ke perangkat Anda

📂 Pulihkan Data (Restore)

Pulihkan data dari file backup .json yang sebelumnya diunduh. Cocok untuk migrasi perangkat atau pemulihan data setelah masalah teknis.

Backup Mingguan

Selalu backup dulu sebelum ganti HP, laptop, atau browser. Restore di perangkat baru.
